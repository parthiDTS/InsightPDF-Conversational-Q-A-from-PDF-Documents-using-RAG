# 📄 InsightPDF – Conversational Q&A from PDF Documents using RAG 🧠

InsightPDF is an AI-powered tool that allows you to ask questions about the content of any PDF file and get accurate answers using **RAG (Retrieval-Augmented Generation)** and **Large Language Models (LLMs)**.

## 🚀 Features

- 🧾 **Upload any PDF** and split it into chunks  
- 🤖 **Ask questions** in natural language  
- 🔍 Uses **RAG (Retrieval-Augmented Generation)** to fetch relevant content  
- 🧠 Answers generated by **LLMs** via OpenRouter  
- 📚 Embeddings powered by HuggingFace's **MiniLM**  
- 💾 Persistent local vector storage with **ChromaDB**

---

## 🛠️ Tech Stack

| Component           | Technology                    |
|---------------------|-------------------------------|
| Backend             | Python                        |
| PDF Loader          | LangChain PyPDFLoader         |
| Text Splitter       | LangChain CharacterTextSplitter |
| Embedding Model     | sentence-transformers/all-MiniLM-L6-v2 |
| Vector Store        | ChromaDB                      |
| LLM API             | DeepSeek via OpenRouter       |
| Chain               | LangChain RetrievalQA         |
| Env Management      | Python Dotenv                 |
